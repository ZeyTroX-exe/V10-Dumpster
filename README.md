# V10-Dumpster (Go)
*A fast Chromium password extractor for Windows*

## Features
 - Extracts saved passwords from Chrome, Edge, Brave
 - Fast and lightweight

## Usage:
```
.\v10-dumpster.exe
```

## Download:
```
git clone https://github.com/ZeyTroX-exe/V10-Dumpster.git

cd V10-Dumpster

go mod tidy

go build -o v10-dumpster .\main.go
```
